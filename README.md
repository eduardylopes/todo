### Executando

1. Clone o repositório

```bash
git clone https://github.com/eduardylopes/todo.git
```

2. Instale as dependências

```bash
npm install
```

3. Instale o ExpoCLI.

```bash
npm i -g expo-cli
```

5. Inicie o Projeto

```bash
npm run start
```
